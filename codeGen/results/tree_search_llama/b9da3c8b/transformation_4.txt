The input should consist of two lines: the first line containing an integer n and the second line containing n space-separated integers
Using a queue data structure to keep track of available officers: In this implementation, a queue is used to store the number of officers available at each time step. When an officer is recruited, they are added to the end of the queue. When a crime occurs and there are no available officers, it can be removed from consideration. The queue ensures that all crimes are processed in chronological order.
