The input should consist of two lines: the first line containing an integer n and the second line containing n space-separated integers
Using a stack data structure to keep track of the number of officers: This approach involves using a stack to store the number of officers recruited at each time step. When an officer is recruited, they are added to the top of the stack. When a crime occurs and there are no available officers, the top element (representing the most recently recruited officers) can be removed from consideration.
